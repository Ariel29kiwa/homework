A checkbox is a small interactive element that allows users to select or deselect an option.
It is typically used in forms and can have various purposes,
such as indicating agreement with terms and conditions,
selecting items from a list, or enabling specific features.

Radio buttons are a type of input element in HTML
that allow users to choose one option from a set of options.
They are typically used when there is a need to select
a single item from a list of choices,
and only one option can be selected at a time within
the same group, as defined by the 'name' attribute.


The email input field, created using the 'input' element with
'type="email,"' allows users to provide their email addresses.
It is typically used in forms for collecting email data from users.
The 'required' attribute indicates that this field must be
filled out before submitting the form.


The number input field, created using the 'input' element with
'type="number,"' allows users to enter numeric values.
It is often used in forms when you need to collect numerical data,
such as age, quantity, or any other numeric input.
The 'min' and 'max' attributes define the allowable
range of values, and the 'step' attribute specifies
the increment or decrement value.
The 'required' attribute indicates that this
field must be filled out before submitting the form.


The search input field, created using the 'input'
element with 'type="search,"' is commonly used for
allowing users to enter search queries on websites.
It provides a user-friendly interface for
searching and typically includes a placeholder
text (e.g., "Search...") to indicate its purpose.
The 'submit' button can be used to initiate the search.

Script Example

This is an example of using the <script> element in HTML.


The <script> element is used to embed or reference JavaScript code within an HTML document.
It can be placed in the <head> or <body> section
of the HTML document. JavaScript code within
the <script> element is executed by the browser
when the page loads or in response to specific events.
It is commonly used for adding interactivity
and functionality to web pages.
